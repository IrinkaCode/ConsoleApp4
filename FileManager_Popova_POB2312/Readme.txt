Название: FileManager
Работа с FileManager

pwd - покажет текущую директорию
tree - выведет дерево (изначально нужно прописать cd "путь")
mkdir(mkdir D:\ - создает папку в указаной директории
ls  (ls D:\papka)- выводит список файлов в указаной директори, показывает последние изменение файла и размер файла
cd (cd D:\papka) - переходит в текущую директорию, чтобы вывести дерево после пишите tree
touch (touch D:\papka) - покажет время и дату последнего изменения
delete и create (delete $$ create D:\papka) создаст или удалит папку(или файл)
